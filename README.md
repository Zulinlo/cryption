# Cryption

## Description
A ReactJs web application that encodes and decodes messages with a variety of cipher options:
- Caesar Cipher
- Affine Cipher
- ROT13 Cipher
- Atbash Cipher
- Baconian Cipher
- Hill Cipher
- Vigenere Cipher

### Executing program

```bash
cd cryption
npm i
npm start
```
